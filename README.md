<h1><b>学生管理系统</b></h1>

![在github部署的开始界面图片](https://cdn.jsdelivr.net/gh/wink-snow/piceeimage/image/MyFistImage.png)

<h2><b>实现的功能：</b></h2>

1. 用户登录的**检测**
2. **循环录入**单个学生信息
3. 显示所有学生信息
4. **排序**后显示所有学生信息
5. 插入学生信息
6. 删除学生信息
7. 查找学生信息

***

<h2><b>信息存储说明</b></h2>

用于存储学生信息的<kbd>stu[MAX_SIZE]</kbd>作为全局变量，存储在内存中的全局区，程序结束后会自动由操作系统清空，无法存留或随时读取。
